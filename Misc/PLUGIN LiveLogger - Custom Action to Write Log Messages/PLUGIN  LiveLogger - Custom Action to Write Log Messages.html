PLUGIN: LiveLogger - Custom Action to Write Log Messages

Requires OS/X Yosemite or later.

ZIP FILE

Please see the Installation Instructions, below.

PURPOSE:

Two Plugin Actions that allow you to write log messages to a text file, and view them more-or-less real-time.

As you can see, log messages include the timestamp of when they were written and the elapsed time between messages, as well as the message itself.

USES:

When you start to think about it, the number of ways you can make use of a feature like this are almost endless. But here's a few uses that come to mind:

Forensic:

Provide an audit trail of what happened when:

Debugging:

Write log messages from various places in a complex macro, to help understand what's really going on. Include the value of variables for even more information:

Optimization:

Use the "Elapsed Time" values to help figure out which sections of a macro are the cause of slow macros.
